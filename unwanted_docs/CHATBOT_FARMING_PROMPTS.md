## üìù TIPS FOR BEST RESULTS

1. **Be Specific**: More details = better advice
   - ‚ùå "My crop has a problem"
   - ‚úÖ "My wheat leaves have brown spots for 3 days"

2. **Include Context**:
   - Crop type
   - Location/region
   - Soil type
   - Season
   - Recent weather

3. **Ask Follow-ups**:
   - If answer unclear, ask again
   - Chatbot remembers conversation

4. **Describe Symptoms Clearly**:
   - Color of leaves
   - Pattern of damage
   - Which parts affected
   - When it started

5. **Ask "Why"**:
   - Not just "what to do" but "why does this work?"
   - Helps you learn
